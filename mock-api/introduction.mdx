## Mock Auth Instructions
<Steps>
  <Step title="First Step">
    Create a MockAPI Industries account, or sign in to an exisiting MAI account.
  </Step>
  <Step title="Second Step">
    Once logged in, go to Settings -> Integrations -> MAI API. Find your auto-generated API Key.
  </Step>
  <Step title="Third Step">
    Include API Key in any requests made to the API. `Authorization: Bearer <api-key>`
  </Step>
</Steps>
