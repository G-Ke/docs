## Built-in Components

### Mermaid

I love Mermaid diagrams. I often use Mermaid entity relationship diagrams to help me visualize data models and schema relationships. Including Meraid diagrams in the documentation easily is one of my favorite features.

The diagram below is an old draft ERD from a project I recently worked on.

```mermaid
erDiagram

    UploadedFile {
        UUID id_external
        string file_name
        file file
        datetime uploaded_at
    }
    FileTextExtraction {
        UUID id_external
        text extracted_text
        datetime processed_at
        int chunk_index
        int page_number
        int chunk_start
        int chunk_end
        json metadata
        vector embeddings_together_m2_bert_80M_2k_retrieval
    }
    Conversation {
        UUID id_external
        datetime created_at
        datetime updated_at
    }
    Message {
        UUID id_external
        string role
        text content
        datetime created_at
        vector embeddings_together_m2_bert_80M_2k_retrieval
    }
    Source {
        UUID id_external
        string name
        text description
        string source_name
        string source_type
        string source_format
        string source_path
        datetime created_at
        datetime updated_at
        json metadata
        json tags
    }
    RecencyFeature {
        UUID id_external
        string name
        text description
        datetime created_at
        datetime updated_at
        json metadata
        json tags
        text content
        datetime timestamp
        float weight
    }
    RelevanceFeature {
        UUID id_external
        string name
        text description
        datetime created_at
        datetime updated_at
        json metadata
        json tags
        text content
        json keywords
        float weight
    }
    DynamicFeatureSet {
        UUID id_external
        string name
        text description
        datetime created_at
        datetime updated_at
        json metadata
        json tags
    }

UploadedFile ||--o{ FileTextExtraction : has
UploadedFile ||--o{ Conversation : associated_with
UploadedFile ||--o{ DynamicFeatureSet : has
Conversation ||--o{ Message : contains
Source ||--o{ UploadedFile : has
Source ||--o{ DynamicFeatureSet : has
DynamicFeatureSet ||--o{ RecencyFeature : has
DynamicFeatureSet ||--o{ RelevanceFeature : has
DynamicFeatureSet }o--|| UploadedFile : belongs_to
DynamicFeatureSet }o--|| Source : belongs_to
```
